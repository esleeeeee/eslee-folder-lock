# eslee폴더잠금기 한계

- eslee폴더잠금기는 암호화 도구가 아닙니다.
- 관리자 권한이 있거나 NTFS ACL을 이해하는 사용자를 완전히 막는 보안 경계가 아닙니다.
- 빠른 모드는 폴더 루트 접근 차단용입니다. 경로와 ACL 구조를 아는 사용자는 일부 상황에서 우회할 수 있습니다.
- 강화 모드는 항목 수가 많은 폴더에서 시간이 오래 걸릴 수 있습니다.
- reparse point, junction, symbolic link 내부로는 안전을 위해 따라가지 않습니다.
- ACL 변경이 중단되면 앱의 복구 기능이나 `eslee폴더잠금기_복구도구.exe`를 사용해야 합니다.
- `data\backups`의 ACL 백업 파일을 삭제하면 자동 복구가 어려워질 수 있습니다.
