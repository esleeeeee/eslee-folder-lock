# eslee폴더잠금기 복구 가이드

eslee폴더잠금기 복구 도구는 메인 앱이 삭제되었거나 중단되었거나 일반 잠금 해제로 복구할 수 없을 때 ACL 백업을 이용해 원래 권한을 되돌리는 독립 실행 도구입니다.

프로그램 파일을 삭제해도 이미 적용된 Windows ACL 규칙은 자동으로 제거되지 않습니다. NTFS 권한은 파일 시스템에 남아 있으므로 복구 도구나 Windows 권한 편집으로 명시적으로 복구해야 합니다.

## 실행 방법

1. 관리자 권한 터미널 또는 탐색기에서 한국어 빌드의 `release\ko\eslee폴더잠금기_복구도구.exe` 또는 영어 빌드의 `release\en\eslee-folder-locker-recovery.exe`를 실행합니다.
2. 복구할 잠금 대상을 선택합니다.
3. 사용할 ACL 백업 파일을 선택합니다.
4. 대상 경로, 백업 파일, 복구 항목 수를 확인합니다.
5. `RESTORE`를 입력해 복구를 실행합니다.

내부 호환성을 위해 빌드 산출물에는 `FolderGate.RecoveryTool.exe`가 함께 남을 수 있습니다. 사용자 안내와 배포 파일명은 언어별 alias를 기준으로 합니다.

## 주의 사항

- 복구 도구는 관리자 권한이 필요합니다.
- 백업 파일은 `data\backups` 아래에 저장됩니다.
- 백업 파일을 삭제하면 자동 ACL 복구가 어려워질 수 있습니다.
- 복구 후에도 문제가 남으면 Windows 보안 속성의 고급 권한 편집 화면에서 ACL을 직접 점검해야 합니다.
